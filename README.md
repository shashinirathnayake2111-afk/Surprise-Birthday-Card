# 🎉 Surprise Birthday Card App

A modern, interactive, and highly emotional digital birthday card built with React, Vite, and Framer Motion. This project is designed as a personalized, premium digital experience featuring glassmorphism aesthetics, 3D animations, and cosmic neon visuals.

## ✨ Features

- 🔒 **Private Lock Screen**: A secure entry point where any 4-digit PIN unlocks the magic.
- 🎬 **Video Intro**: A sleek, cinematic intro sequence setting a warm, celebratory mood.
- 📸 **Memories Carousel**: An interactive, infinite 3D photo carousel wrapped in glowing glassmorphism to showcase your best moments together.
- 💌 **Heartfelt Message Page**: A beautiful typographic section that delivers your personalized birthday wishes.
- 🎁 **Surprise Hub**: The ultimate interactive playground!
  - **Friendship Grid**: 9 animated 3D "Galaxy Boxes" that flip to reveal why your bestie is amazing.
  - **Surprise Gift Selector**: An interactive game where they choose one special gift from 5 options. Once picked, a suspense timer counts down, reveals the choice with confetti, and after 20 seconds, reveals the rest of the unselected gifts!
  - **WhatsApp Integration**: A sleek neon button to instantly share the chosen gift via WhatsApp.
- 🐼 **Cute Floating Pandas**: Animated chibi pandas that peek and wave across the screen.

## 🚀 Technologies Used

- **React 18** (Functional components, Hooks)
- **Vite** (Next-generation, fast frontend tooling)
- **Framer Motion** (For fluid, spring-based animations & page transitions)
- **Lucide React** (Beautiful SVG icons)
- **Canvas Confetti** (For celebratory particle effects)
- **CSS3** (3D transforms, glassmorphism, flexbox, CSS variables, keyframe animations)

## 🛠️ How to Run Locally

1. **Clone the repository** (if applicable) or navigate to the project directory:
   ```bash
   cd "Surprise Card"
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser**:
   Visit the local URL provided in the terminal (usually `http://localhost:5173/`).

## 🎨 Design System

- **Color Palette**: Dark cosmic backgrounds paired with vibrant neon gradients (Pink, Purple, Cyan, Gold).
- **Typography**: Uses modern fonts like `Outfit` and `Playfair Display` for a premium look.
- **Glassmorphism**: extensive use of `backdrop-filter: blur()`, semi-transparent backgrounds, and soft white inset borders to create frosted glass interfaces.

## 📱 Responsiveness

The app is fully responsive and optimized for Desktop, Tablet, and Mobile devices. 3D geometry and font sizes dynamically adjust via CSS media queries to ensure perfect rendering on smaller screens.
