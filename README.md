# 🎂 Bolaaa Badooo's Birthday Surprise 🎈

A high-end, multi-stage interactive digital experience created for **Nalaka**'s birthday. This project combines cutting-edge web technologies with a deeply personalized "Cyber-Luxury" aesthetic to create a memorable birthday celebration.

## 🌟 Key Features

### 1. Interactive Love Letter 💌
- **3D Envelope Animation**: A realistic envelope reveal where the letter slides out gracefully.
- **Symmetric Special Notes**: Floating advice and wishes that appear with a glow effect as the letter opens.
- **Responsive Layout**: Designed to feel like a premium physical card in a digital space.

### 2. Memories Page (Photo Rain) 📸
- **Immersive Photo Rain**: Over 21+ personalized WhatsApp images falling through the screen in a dense, high-opacity "rain" effect.
- **Dynamic Animations**: Photos rotate and scale randomly as they fall, creating a cinematic memory lane.

### 3. Surprise Hub (The Grand Finale) 🌌
- **Love Galaxy**: A 3D orbital layout of 9 reasons why Nalaka is loved. Replaces traditional grids with a dynamic cosmic arrangement.
- **Magic Words**: A randomized compliment generator featuring holographic glassmorphism cards.
- **Dream List**: A neon-accented bucket list of future shared adventures.
- **Panda Family**: Three unique, animated panda stickers (Heart, Waving, and Peeking) that accompany the journey with friendly messages.

## 🛠️ Technical Stack
- **Framework**: React 18
- **Animations**: Framer Motion (Lifecycle & Entrance animations)
- **Visual Effects**: Canvas Confetti
- **Icons**: Lucide React
- **Styling**: Vanilla CSS with modern tokens (Glassmorphism, Neon Glows, Radial Gradients)

## ✨ Aesthetics
- **Theme**: Cyber-Luxury / Cosmic
- **Palette**: Deep Space Black, Neon Pink, Gold, and Cyan accents.
- **Typography**: Playfair Display (Elegant) & Outfit (Modern/Tech).

## 🚀 Getting Started
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to launch the surprise locally.
